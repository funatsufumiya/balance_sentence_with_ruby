# 声優統計コーパス 音素バランス文 ルビ付き

声優統計コーパスの音素バランス文に、ルビを付けたものです。

ルビ振りには[でんでんエディター](https://edit.denshochan.com/)を使っており、txtファイルをでんでんエディターに読み込ませるとHTMLに変換できます。
